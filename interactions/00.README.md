# Interactions

This directory contains the 108 LLM interactions analyzed in the study “Large Language Models in Industrial Systems Engineering Learning: A Pilot Study on Solving Production Systems Problems.”
Interactions are in Italian.
Each interaction is available in two forms:

1. a static PDF copy stored in this repository;
2. a link to the original shared conversation, listed in the Excel index.

## Directory structure

The PDF files are organized according to the LLM configuration used in the experiment:

```text
interactions/
├── CGPT/
├── GemV/
├── GemR/
├── 00.README.md
└── Links to interactions.xlsx
```

The folders contain:

* `CGPT/`: interactions generated using ChatGPT;
* `GemV/`: interactions generated using the fast-response version of Gemini;
* `GemR/`: interactions generated using the reasoning-oriented version of Gemini.

Each folder contains 36 interactions, corresponding to the nine exercises tested under four prompting conditions.

## File naming convention

The PDF files follow this naming convention:

```text
[interaction number]. [exercise code], [prompt condition].pdf
```

For example:

```text
001. CT1, P1.pdf
```

In this example:

* `001` is the unique interaction number;
* `CT1` is the exercise identifier;
* `P1` is the prompting condition;
* the folder in which the file is stored identifies the LLM configuration.

The interaction number corresponds to the identifier reported in the Excel index.

## Prompting conditions

The following prompting conditions were used:

* `P1`: zero-shot baseline, containing only the exercise text;
* `P2`: Chain-of-Thought instruction;
* `P3`: professional role assignment;
* `P4`: one-shot prompting with a worked example.

The complete formulation of the prompting conditions is available in the `prompts` directory of the repository.

## Excel index

The file `Links to interactions.xlsx` provides an index of all 108 interactions.

For each interaction, the spreadsheet reports the relevant experimental identifiers and a link to the original shared conversation.

The links are included to facilitate direct inspection of the conversations on the original platforms. The PDF files stored in the three model folders serve as static archived copies, since the continued availability of external shared links cannot be guaranteed.

## Preservation of the original outputs

The interaction contents were preserved as generated by the respective LLM interfaces. The PDFs reproduce the prompts and responses used in the study and were not edited to correct calculation, reasoning, formatting, or language errors.

Each exercise was submitted in an independent conversation so that the model could not rely on contextual information from previous interactions.
