# Exercises

This directory contains the nine production-systems exercises used in the experimental study “Large Language Models in Industrial Systems Engineering Learning: A Pilot Study on Solving Production Systems Problems.”

The exercises cover three core topics in Industrial Systems Engineering:

* theoretical production capacity (CT);
* bottleneck identification (BN);
* required material input rate (MP).

For each topic, three exercises with increasing levels of difficulty were developed, resulting in a total of nine exercises.

## File contents

Each exercise is identified by a short code, such as:

```text
CT1
```

Each file contains:

1. the complete exercise statement submitted to the LLMs;
2. the reference solution used to determine the expected result;
3. the relevant calculations and logical steps required to solve the exercise.

The exercise codes correspond to those used in the interaction filenames, the Excel interaction index, and the experimental dataset.

For example, an interaction file named:

```text
001. CT1, P1.pdf
```

refers to exercise `CT1` under prompting condition `P1`.

## Language

The exercise statements and reference solutions are written in Italian.

The original Italian wording has been preserved because these are the exact exercise texts used during the experiment. No English translation was introduced into the prompts submitted to the models.
