# Large Language Models in Industrial Systems Engineering Learning

This repository contains the supplementary materials associated with the study:

**“Large Language Models in Industrial Systems Engineering: A Pilot Study on Solving Production Systems Problems.”**

The study investigates the reliability of Large Language Models (LLMs) in solving quantitative production-systems problems and examines how different prompting conditions influence their performance.

## Repository contents

The repository is organized into two main directories:

```text
.
├── exercises/
└── interactions/
```

### `exercises/`

This directory contains the nine production-systems exercises used in the experiment.

For each exercise, the directory provides:

* the original exercise statement;
* the reference solution;
* the calculations and logical steps used to evaluate the LLM responses.

The exercises and reference solutions are written in Italian, preserving the original wording used during the experiment.

Exercise identifiers, such as `CT1`, correspond to the codes used in the interaction filenames and in the Excel index.

Further information is available in the README file inside the `exercises` directory.

### `interactions/`

This directory contains the 108 independent LLM interactions analyzed in the study.

The interactions are organized into three subdirectories according to the tested LLM configuration:

* `CGPT/`: ChatGPT interactions;
* `GemV/`: interactions generated with the fast-response version of Gemini;
* `GemR/`: interactions generated with the reasoning-oriented version of Gemini.

Each interaction is preserved as a PDF file.

The directory also contains an Excel file providing an index of all interactions and links to the corresponding shared conversations on the original platforms.

Further information about the file naming convention and directory structure is available in the README file inside the `interactions` directory.

## Language

The exercises, prompts, and reference solutions are written in Italian.

The language and wording of the original experimental materials have been preserved to ensure consistency with the interactions analyzed in the study.

## Preservation and access

The PDF files stored in the repository provide static copies of the analyzed interactions.

The Excel index also contains links to the original shared conversations. These links are provided for convenient inspection, but their continued availability depends on the external platforms.

The archived PDFs should therefore be considered the stable copies of the experimental interactions.
